# Eleccion_users

Esta carpeta es donde puedes guardar manualmente los archivos `Usuarios.txt`
que la gente descargue desde el formulario "Pide la colonia que quieras ver".

Importante: GitHub Pages no tiene servidor, así que la web no puede escribir
aquí sola. El botón "Descargar registro" de la página genera un
Usuarios.txt con lo guardado en el navegador de cada visitante; si quieres
centralizarlo, puedes:

1. Pedir a cada persona que te reenvíe el Usuarios.txt descargado, o
2. Conectar el formulario a un servicio externo gratuito (Formspree,
   Google Forms + Sheets, Airtable...) que sí guarda los envíos de todo el
   mundo en un único sitio. Hay un ejemplo comentado en funcion.js, dentro
   de la función guardarEleccionUsuario().
